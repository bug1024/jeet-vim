##Installation
    git clone https://github.com/bug1024/jeet-vim.git
    git clone https://github.com/gmarik/vundle.git jeet-vim/vim/bundle/vundle/
    ln jeet-vim/vimrc ~/.vimrc
    ln jeet-vim/vimrc.bundles ~/.vimrc.bundles
    ln jeet-vim/vim/ ~/.vim
