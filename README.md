# jeet-vim
Vim configuration for terminal

## My environment
* ubuntu 14.04
* OS X EI Captian 10.11.6
* vim 7.4

## Installation
    curl https://raw.githubusercontent.com/bug1024/jeet-vim/master/install.sh | bash

## Key mappings
    ,2 NerdTree
    ,3 Tagbar
    ,6 Syntax on|off
    ,7 Clear tab
    ,8 Clear space
    ,p Search file
    ,f Files Recent Open
