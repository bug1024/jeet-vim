##jeet-vim
Vim configuration for PHP developer

##My environment
* ubuntu 14.04
* vim 7.4

##Installation
    sh install.sh
    
##Key mappings
    <F2> NerdTree
    <F3> Tagbar
    <F6> Syntax on|off
    <F7> Clear tab
    <F8> Clear space
    ,p Search file
    ,f Files Recent Open
    ,num Go to tab quickly, num = 1~9
